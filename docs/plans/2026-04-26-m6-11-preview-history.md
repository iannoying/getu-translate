## Cross-cutting decisions (settle here)

- **Iframe sandbox:** `<iframe sandbox="allow-same-origin" src="...">`. The HTML is generated by us, so no need for `allow-scripts`. The HTML is self-contained (inline CSS, no JS).
- **Download URL strategy:** signed R2 presigned URL (1-hour expiry) returned by `documentDownloadUrl`. This avoids streaming through the Worker. Use `aws4fetch` (already in deps) for signing.
- **Polling stop condition:** stop polling after 5 minutes total (sanity timeout); show "still processing ‚Äî check back later" message.
- **Retry flow:** Retry button calls `documentRetry` (NEW orpc procedure) that creates a new job pointing to the same `source.pdf` key. M6.11 includes this.
