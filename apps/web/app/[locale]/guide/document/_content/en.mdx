# PDF Document Translation

Upload any PDF and receive a fully translated bilingual output — side-by-side
HTML view and alternating Markdown. Powered by the same AI models as the text
translator.

---

## Use Cases

- **Academic papers** — translate research articles while keeping structure
- **Contracts &amp; legal docs** — review foreign-language agreements
- **Technical documentation** — manuals, specifications, datasheets
- **Books &amp; long reports** — multi-page documents translated in one job

---

## How It Works

1. **Upload** — drag and drop a PDF or click to select a file
2. **Extract** — the server parses the PDF and extracts text by page
3. **Chunk** — text is split into translation-friendly segments
4. **Translate** — each chunk is sent to your chosen AI model
5. **Output** — results are assembled into bilingual HTML and Markdown

## Output Formats

**Side-by-side HTML** — original and translated text displayed in two columns,
page by page. Download as a self-contained HTML file and view offline.

**Alternating Markdown** — original paragraph followed immediately by its
translation, repeated throughout the document. Ideal for copying into note
apps or further editing.

## Known Limitations

- **Scanned PDFs** (image-only, no text layer) are not supported. The extractor
  requires selectable text. OCR support is planned for a future release.
- Tables and complex layouts may lose some formatting in the output.
- Very large files may take several minutes to process.

---

## FAQ

**Which AI models are available for document translation?**
The same models as text translation. Free users can use Google, Microsoft,
LibreTranslate, and GPT-4o mini. Pro users have access to all 11 models.

**How long does translation take?**
Typically 1–3 minutes for a 10-page document, depending on the model and server
load. You can close the tab and come back — results are stored for your session.

**Can I translate a PDF from a URL?**
Yes. Append `?src=<url>` to the `/document` page URL and the system will
fetch and translate it automatically.

**Are my PDF files stored permanently?**
Source files are deleted after processing. Output HTML/MD files are retained
according to your tier's retention policy (see Quota Rules above).
